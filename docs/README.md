# 문서 디렉토리

## 📁 디렉토리 구조

```
docs/
├── architecture/        # 아키텍처 및 배포 다이어그램
├── guides/             # 가이드 문서 (설정, 사용법)
├── troubleshooting/    # 트러블슈팅 가이드
├── analysis/          # 분석 및 평가 문서
└── reports/           # 리포트 문서
```
